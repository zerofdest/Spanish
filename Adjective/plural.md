### 规则1：多数以元音结尾（尤其非重读元音）→ 加 **-s**

* blanco → blancos
* grande → grandes（`-e` 也通常加 `-s`）

RAE DPD 对“以非重读元音或重读 -e 结尾”给出加 **-s** 的通则。
#### 以重读 **-í / -ú** 结尾 → 常有双形式（-es / -s）

RAE DPD 与基础语法都指出：这类常常允许两种复数，规范语体里往往偏好 **-es**；但实际使用中 **-s** 也可能常见，尤其借词或口语词。

* carmesí → carmesíes / carmesís
* hindú → hindúes / hindús（常偏好 hindúes）

如果你是考试/正式写作，优先用 **-es**，通常更稳。
### 规则2：多数以辅音结尾 → 加 **-es**

尤其常见于结尾为 `-l, -n, -r, -d, -z` 的词；`-z` 变 `c` 再加 `-es`。RAE 文法条目明确列出。

* azul → azules
* útil → útiles
* español (作形容词) → españoles
* feliz → felices（z → c）

这是学习者最常用、也最稳定的一条。

#### 以 **-s / -x** 结尾时，先看重音位置（常见难点）

RAE 基础语法总结得很清楚：

* **重音在最后（agudas）或单音节** → 多用 **-es**
* **非重读最后音节（如 llanas/esdrújulas）** → 往往**不变形**。

虽然该条常见于名词，但 DPD 说明这是名词和形容词共同的复数规则体系。

典型类比（帮助你判断）：

* compás → compases（重音在末）
* gratis（形容词常见）→ 通常不变（单复数同形）

这里实际用法会受到词源、词类化程度、地区差异影响，因此要容许少量波动。
