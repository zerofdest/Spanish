## 主规则
### A. 词尾是**非重读元音**（通常就是不带重音符号的 a/e/i/o/u）

**直接 +s**

* casa → casas
* libro → libros
* tribu → tribus

这是最稳定、最高频的一条。RAE 明确把它作为一般规则。

### B. 词尾是**辅音**（一般情况）

**直接 +es**（尤其是 -l/-r/-n/-d/-j/-z 等结尾）

* árbol → árboles
* papel → papeles
* reloj → relojes
* pared → paredes

RAE 的基本规则如此总结。

### C. 词尾是 **-z**

**z → c + es**

* luz → luces
* pez → peces

这是西语里非常规律、几乎“零例外”的一类。

## “额外判断”的分支
### 2.1 词尾是**重读元音** -á / -é / -ó

**通常 +s**
* café → cafés
* bebé → bebés
* sofá → sofás

RAE 明确给出：-á/-é/-ó 的复数用 -s。

**额外情况的“多寡/命中范围”**：
这类规则本身很稳；真正的“例外词”非常少（RAE点名的例外属于极小集合，如 faralá → faralaes 等）。可近似理解为>99% 都按 +s。
- faralá → faralaes
- albalá → albalaes
- no（作名词用，“一个/多个‘不’”）→ noes
### 2.2 词尾是**重读 -í / -ú**
这里是最典型的“要多判断一步”的地方：
**两种复数往往都可：-es 或 -s**，但**书面/较规范语体常偏好 -es**，尤其是国籍/族名。
* bisturí → bisturíes / bisturís
* bambú → bambúes / bambús

**额外情况**：
* **gachí → gachís**
* **pirulí → pirulís**
* **popurrí → popurrís**
* **champú → champús**（部分地区也见 champúes）
* **gurú → gurús**（也可见 gurúes）
* **haikú → haikús**
* **menú → menús**（也可见 menúes）
* **travestí → travestís**
* **tutú → tutús**
* **vermú → vermús**


### 2.3 词尾是 **-y（前面是元音）**

**传统/一般：+es**

* rey → reyes
* ley → leyes

**额外情况多寡**：
“-y 结尾名词”在日常词汇里相对少；规则本身挺稳。真正麻烦更多来自外来词的拼写与适应程度（如某些外来词复数写法会更像“把 y 当 i”来处理），属于“外来词分支”问题。
* **gay → gais**
* **jersey → jerséis**
* **espray → espráis**
* **yóquey → yoqueis**

### 2.4 词尾是 **-s 或 -x：是否重读（aguda）决定**

这是另一个高频坑点，因为很多常用词恰好落在这里（lunes, crisis, análisis…）。

* 如果是**单音节**或**重读在最后一音节（aguda）**：**+es**

    * tos → toses
    * compás → compases
    * fax → faxes
    * **compás → compases**
    * **francés → franceses**（既可作形容词也常作名词“法国人/法语”）
    * **malpaís → malpaíses**
* 否则（多为**平音 llana**或**esdrújula**）：**形式不变**（复数靠冠词/一致关系体现）
  * la crisis → las crisis
  * el lunes → los lunes
  * el tórax → los tórax
  * el fórceps → los fórceps
* “即使是单音节也不变”的词级例外
    * **dux → dux**
    * **ex → ex**

### **“重读位置通常不变；音节数变了，按重音规则重新决定 tilde。”**

#### 常见类型 1：加 -es 后变成 esdrújula，需要补 tilde

* examen → exámenes
* canon → cánones
* crimen → crímenes
* mitin → mítines

#### 常见类型 2：原来是 aguda 带 tilde，复数后变 llana，tilde 可能消失

* almacén → almacenes
* refrán → refranes
* gorrión → gorriones

#### 少数“重音位置真变化”的词（要单独记）

* carácter → caracteres
* régimen → regímenes
* espécimen → especímenes
