## -o -a
**词尾 -o 的名词约 99.9% 为阳性**
**词尾 -a 的名词约 96.3% 为阴性**

## 阴性后缀 -triz, -ez, -dad, -ción, -sión, -tud

* **-ción / -sión** → 几乎总是阴性（如 *nación, televisión*）
* **-dad / -tad** → 高度稳定阴性（如 *ciudad, libertad*)
* **-tud** → 通常阴性（如 *actitud*)
* **-umbre** → 通常阴性（如 *costumbre*)
* **-triz** → 阴性（如 *actriz, cicatriz* 中 *-triz* 结构）

适用百分比：
* **-ción / -sión**：约 **95–99%+**（教学上几乎可当稳定阴性）
* **-dad / -tad / -tud**：约 **90–99%**（高度可靠）
* **-umbre**：约 **85–95%**（可靠，但词量较少）
* **-triz**：在常见词中可视作 **很高可靠**（接近“词形即提示阴性”）


## 阴性名词前用 “el” 的情况
适用于“阴性 + 重读 /a/ 开头 + 单数 + 冠词紧邻”的结构，实际命中可视作**接近规则性（>95%）**。
如：**el agua, el águila, el aula, el hacha**，但仍是阴性名词。RAE DPD 还说明：

* 只在冠词紧贴名词时出现
* 中间插入词就恢复 **la**（如 *la mejor agua*）
* 指示词应说 **esta agua**，不是 *este agua* 


## 通性名词（el/la + 同一名词形式）

如职业/身份类中不少词：*el/la estudiante, cantante, agente* 等。MDPI 页面也特别提到很多这类 -e 结尾人物名词。
CVC 也列出 *el/la guía, el/la colega, el/la testigo* 等。

**判断法**

* 这类名词的“词形”不直接告诉你指称对象的自然性别
* 要看**冠词/限定词/上下文**

**适用范围（粗略）**

* 对“表示人、职业、角色”的若干结尾（尤其 -ista、很多 -nte/-e），把它当作“通性候选”很有用
* 但不能把所有 -e/-nte 都当通性，因此规则适用范围大约是**特定语义类内高、跨类低**
